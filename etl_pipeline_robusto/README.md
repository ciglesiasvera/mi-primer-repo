# Pipeline ETL Robusto con Logging y Transacciones

Este proyecto implementa un pipeline ETL completo en Python que incluye:

- Logging estructurado
- Manejo robusto de errores
- Reintentos en extracción
- Validaciones de datos
- Carga transaccional en SQLite
- Reporting final

## Ejecución

```bash
python etl_pipeline.py
